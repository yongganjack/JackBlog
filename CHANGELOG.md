# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses semantic versioning where practical.

## [0.1.0] - 2026-07-12

### Added

- Initial Astro + Tailwind static blog implementation.
- Added 8 routes: home, archive, post detail, tags, projects, about, search, and 404.
- Added shared components for layout, header, footer, article cards, project cards, tags, pagination, TOC, code blocks, search, and theme toggle.
- Added local SVG assets for hero illustration, covers, project thumbnails, avatar, and 404 page.
- Added GitHub-ready documentation and templates.
