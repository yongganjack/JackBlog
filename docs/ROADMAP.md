# Roadmap

## Version 0.1

- Astro + Tailwind 静态博客基础页面。
- 文章、标签、项目、关于、搜索和 404 页面。
- 本地静态数据和 SVG 素材。
- GitHub-ready 文档。

## Version 0.2

- 将文章迁移到 Astro Content Collections。
- 添加动态文章详情路由。
- 添加 RSS 和 Sitemap。
- 增加真实文章 Markdown/MDX 示例。

## Version 0.3

- 接入 Pagefind 本地搜索。
- 接入 Giscus 评论。
- 增加阅读进度条。
- 优化 SEO Open Graph 图片。

## Later

- 多语言支持。
- 系列文章。
- Umami 或其他轻量统计。
- 专注阅读模式。
- PWA 和离线阅读。
