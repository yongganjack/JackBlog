# Code of Conduct

## Our Pledge

我们希望这个项目保持友好、开放和建设性的协作氛围。参与者应尊重不同经验水平、背景和观点。

## Expected Behavior

- 使用友善、清晰和尊重的语言。
- 针对问题和方案讨论，避免人身攻击。
- 接受建设性反馈，也愿意解释自己的设计取舍。
- 在发现问题时提供可复现信息。

## Unacceptable Behavior

- 侮辱、骚扰、歧视或威胁。
- 发布他人的隐私信息。
- 恶意刷屏、引战或破坏项目维护。
- 明知有害仍提交恶意代码或误导性内容。

## Enforcement

维护者可以删除不当内容、关闭讨论、限制参与者互动，或在严重情况下阻止继续参与项目。
