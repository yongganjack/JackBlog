# Security Policy

## Supported Versions

当前项目处于早期版本，安全修复优先覆盖最新版本。

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a Vulnerability

如果你发现安全问题，请不要直接公开利用细节。可以通过以下方式联系维护者：

- GitHub Security Advisory: 如果仓库已启用 Security Advisories，优先使用该方式。
- Maintainer Email: 上传公开仓库前，请在这里替换为你的安全联系邮箱。

报告中建议包含：

- 受影响的页面、组件或依赖。
- 复现步骤。
- 影响范围。
- 你建议的修复方向。

## Scope

当前项目是静态站点模板，主要安全关注点包括：

- 第三方依赖漏洞。
- 外链脚本或分析工具配置。
- 未来接入评论、搜索、表单或 CMS 后产生的数据输入风险。
